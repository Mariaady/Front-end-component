# PawTrip - Frontend

Frontend de PawTrip, aplicación para viajeros con mascotas.

## Tecnologías
- React, React Redux, React Router, Vite, React Icons

## Instalación y Ejecución

1. Clonar repo
2. Instalar dependencias: `npm install`
3. Ejecutar: `npm run dev`
